# ❤️ Valentine Proposal Website

A fun and interactive Valentine-style proposal website built using HTML, CSS, and JavaScript.

This project features a playful proposal page with animated buttons, growing effects, falling hearts, and a romantic final message.

## ✨ Features

- Beautiful pastel-themed UI
- Interactive Yes and No buttons
- "Yes" button grows whenever "No" is clicked
- Romantic popup screen with GIF
- Falling heart animation
- Responsive design for mobile and desktop
- Hinglish messages for a personal touch

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
